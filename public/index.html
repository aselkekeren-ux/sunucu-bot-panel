Özel JavaScript Modu

Bot sunucuya bağlandığında çalışacak özel bir JS dosyası yükleyebilirsiniz. bot, logBot değişkenleri kullanılabilir.

Seçilmedi
(Bot yeniden başlatıldığında etkin olur)

Örnek Script:

// Bot 5sn'de bir zıplar
setInterval(() => {
  bot.setControlState('jump', true);
  setTimeout(() => bot.setControlState('jump', false), 300);
  logBot('Zıplandı!');
}, 5000);

// Oyuncu "merhaba" yazarsa cevap ver
bot.on('chat', (user, msg) => {
  if (user === bot.username) return;
  if (msg.toLowerCase().includes('merhaba'))
    bot.chat('Merhaba ' + user + '!');
});
